Geek's Candy
Accuracy: 0.0% Submissions: 0 Points: 100

Geek is having n different candies bucket and each bucket contain bucket[i] candies(bucket[i] >=1) and one day he wonder to find two buckets such that the sum of absolute difference between each bucket to one of the two chosen bucket is minimal.

Example 1:

Input:
n = 3
bucket = [1,3,5]
Output:
2
Explanation:
Geek can pick two bucket 1 and 4,
so required sum = |1-1| + |3-4| + |5-4| = 0 + 1+1 = 2

Example 2:

Input:
n = 4
bucket = [3,2,5,11]
Output:
3
Explanation:
Geek can pick 3 and 11 to get required sum = 3.

Your Task:
You don't need to read input or print anything. Complete the function Solve() that takes parameter n and bucket of candies and returns the required sum.

Constraint:
2 <= n <= 5 * 103
1 <= bucket[i] <= 105 

Expected Time Complexity: O(n * log(n))
Expected Space Complexity: O(1)
